## Operadores de Comparación en Python

| Operador | Comparador    | Ejemplo |
|----------|---------------|---------|
| ==       | Igual         | x == y  |
| !=       | No Igual      | x != y  |
| \>       | Mayor que     | x > y   |
| <        | Menor que     | x < y   |
| \>=      | Mayor igual a | x >= y  |
| <=       | Menor igual a | x <= y  |
